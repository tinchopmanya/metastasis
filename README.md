# metastasis

Repositorio de investigación sobre metástasis, con foco actual en cáncer colorrectal y metástasis hepática.

## Estructura actual
- `Investigacion.md`: log central y reglas del sistema de investigación.
- `InvestigacionMapa.md`: mapa cronológico de olas y archivos asociados.
- `Conlusion.md`: conclusión dinámica vigente.
- `InvestigacionSobre*.md`: investigaciones extensas.
- `ResumenInvestigacionSobre*.md`: resúmenes fuertes de cada investigación.

## Estado actual
- Ola 001: panorama sobre metástasis en cánceres y oportunidades con IA.
- Ola 002: cáncer colorrectal y metástasis hepática.

## Flujo de trabajo
1. Abrir una nueva ola de investigación.
2. Crear un archivo `InvestigacionSobreTema.md`.
3. Crear un archivo `ResumenInvestigacionSobreTema.md`.
4. Actualizar `InvestigacionMapa.md`.
5. Reescribir `Conlusion.md`.
